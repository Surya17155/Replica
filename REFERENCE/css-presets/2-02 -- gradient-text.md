# Gradient Text

```css
.hero-heading {
  background: linear-gradient(to right, #FF6B6B, #FFE66D, #4ECDC4);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}

/* For dark themes with purple/magenta gradient */
.accent-gradient {
  background: linear-gradient(135deg, #B600A8 0%, #7621B0 100%);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}
```

**Tailwind:** `bg-gradient-to-r from-[#B600A8] to-[#7621B0] bg-clip-text text-transparent`
